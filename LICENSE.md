## 🛠️ Troubleshooting Common Issues

- ❌ **Script does not execute**
  The executor may not be attached properly. Close Roblox, reopen it, rejoin the Football Fusion 3 lobby, and re-attach before pasting the script again.
  **Solution:** Make sure your executor is on the latest version and that you attached *after* the game fully loaded.

- ❌ **GUI does not appear after execution**
  A conflicting script or outdated executor build can block the interface from rendering.
  **Solution:** Clear your executor's cache folder, restart the app, and run the Football Fusion 3 Script on a fresh session.

- ❌ **Hitbox Expander has no visible effect**
  The `HitboxSize` value in your config may be set too low, or the Football Fusion Hitbox Script module is toggled off.
  **Solution:** Open the settings panel with `F8` and confirm the Hitbox module is enabled with a value of at least `10`.

- ❌ **Lag spikes during ESP usage**
  ESP draws highlight boxes for every player each frame, which can stress weaker GPUs.
  **Solution:** Disable `ShowDistance` in the ESP config or turn ESP off on devices with integrated graphics.

- ❌ **Auto Catch triggers too early**
  The `Range` value might be set higher than intended, causing catches outside the realistic zone.
  **Solution:** Lower `AutoCatch.Range` to `18` or `20` in your config table for better timing.
